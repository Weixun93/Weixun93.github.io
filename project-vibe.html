汗水日記

Vibe Coding React and Database Full-Stack

以 React 為前端的健身日記應用程式，用戶可以記錄訓練內容、追蹤進度，並透過圖表視覺化分析健身成果。

專案簡介

汗水日記是一款專為運動分享者設計的應用程式，期望大家可以持之以恆激勵彼此。用戶可以輕鬆記錄每日的訓練內容、時間、難易度與重複次數，並追蹤長期運動次數與進度。系統提供個人化的健身建議，並支援好友互動功能，讓健身不再孤單。

功能特色

  • 訓練記錄與追蹤

    用戶可以記錄各項運動的詳細數據，包括重量、時間、重複次數，並自動追蹤達成日期與進度。

  • 數據視覺化分析

    透過圖表展示運動成果，並提供個人化建議，可以透過 AI 問答機器人得到關於訓練的建議與指導。

  • 社交互動功能

    用戶可以關注好友、分享訓練成果，並互相鼓勵，提升健身動力。

技術架構

React Node.js PostgreSQL Render

前端採用 React 框架開發，提供流暢的用戶體驗。後端使用 Node.js 與 Express 處理 API 請求，資料庫採用 PostgreSQL 儲存用戶數據與訓練記錄。應用程式部署在 Render 平台上，確保高可用性。

開發挑戰與解決

在開發過程中，最大的挑戰在於數據同步與用戶體驗優化。透過實時數據更新與優化的 UI 設計，確保用戶在記錄訓練時的流暢性。此外，也注重資料安全，採用加密儲存用戶個人資訊。
