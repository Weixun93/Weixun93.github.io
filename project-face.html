人臉辨識打卡系統

Computer Vision System Design OpenCV

基於 OpenCV 的即時人臉辨識解決方案，實現無接觸式自動化考勤管理。

專案簡介

傳統的打卡方式（如卡片、指紋）存在代打卡或衛生疑慮。本專案開發了一套低成本、高效率的人臉辨識打卡系統，適用於中小型企業或教室場景。系統能即時捕捉鏡頭畫面，自動偵測並識別註冊人員，完成打卡記錄。

功能特色

  • 即時人臉偵測與識別

    應用 Haar Cascade 分類器進行快速人臉偵測，並結合 LBPH (Local Binary Patterns Histograms) 或深度學習模型進行特徵比對與身份識別。

  • 自動化考勤記錄

    識別成功後，系統自動將員工 ID、姓名與打卡時間寫入 JSON 格式的檔案，方便後續匯出與時程計算薪資。

  • 防偽機制（基礎）

    透過活體檢測（Liveness Detection）的基礎邏輯，防止使用照片或影片進行惡意欺騙。

技術細節

Python OpenCV NumPy JSON

核心演算法基於 OpenCV 函式庫開發。系統流程包含：影像前處理（灰階化、直方圖均化）、特徵提取、模型訓練與即時推論。資料儲存採用輕量化的 JSON 格式，便於系統移植與快速部署。

挑戰與優化

開發過程中最大的挑戰在於光線變化對辨識準確率的影響。透過加入影像增強預處理步驟，顯著提升了在昏暗或背光環境下的辨識率。此外，也優化了程式碼結構，確保在一般筆記型電腦上也能達到流暢的 FPS 表現。
