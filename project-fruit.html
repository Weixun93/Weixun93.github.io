果然會辨識 (畢業專題)

AI / CV App Dev System Integration

榮獲畢業專題最佳人氣獎。整合深度學習與行動應用，實現即時水果種類與甜度辨識。

[ 專案展示圖片/影片區域 ]

專案背景與目標

在現代農業與消費者選購水果的過程中，如何快速判斷水果的種類與品質（如甜度）一直是一個有趣的課題。本專案旨在開發一套整合系統，讓使用者透過手機鏡頭拍攝水果，即可獲得其種類資訊與預估甜度，展現 AI 在日常生活中的應用潛力。

核心功能

  • 即時種類辨識

    利用 YOLOv11 模型進行物件偵測，能快速準確地在複雜背景中框出水果位置並辨識種類。

  • 甜度預測分析

    結合 VGG-16BN 進行遷移式學習，分析水果外觀特徵（如顏色、紋理）與甜度的關聯，提供甜度等級預測。

  • 行動端整合

    開發 Android 應用程式，透過手機相機擷取影像，並經由 FastAPI 與後端 AI 模型進行即時通訊。

技術架構

Python PyTorch YOLOv11 VGG-16BN FastAPI Android Studio (Java/Kotlin)

系統架構採用前後端分離設計。後端伺服器運行深度學習模型，負責核心的推論工作；前端 Android App 負責影像採集與結果呈現，兩者透過 RESTful API 進行資料交換，確保系統的即時性與擴充性。

成果與反思

本專案在畢業專題展中獲得了「最佳人氣獎」，驗證了其應用場景的吸引力。在開發過程中，我們克服了模型在移動裝置上推論延遲的問題，並透過資料增強（Data Augmentation）提升了模型在不同光源下的泛化能力。未來可進一步結合 AR 技術，提供更直覺的互動體驗。
